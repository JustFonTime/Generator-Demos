# What is This?

This tool is meant to demonstrate basic noise generation using the FastNoiseLite package created for Unity implementation. 

The output of this tool is a simple map layout using the type of noise specified by users (available noises from the imported package).

Link to the package repository: https://github.com/shniqq/FastNoiseLite-Unity



# How to Use:

1. Select the type of noise you would like to use in the generation.

2. Press Generate.

3. Press [R] to regenerate the output with the same parameters. Press [ESC] to go back to prompt screen to alter parameters.
